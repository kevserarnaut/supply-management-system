# Supply Management System

A Java-based Supply Management System designed to manage factories, markets, customers, products, and inventory operations through an object-oriented and MVC-inspired architecture.

## Overview

This project was developed as a real-world business management simulation to improve software engineering, backend development, and object-oriented programming skills.

The system focuses on modular design, clean code organization, and management operations between different business entities.

## Features

* Factory management
* Market management
* Customer management
* Product and inventory tracking
* Producer management
* Byproduct handling system
* CRUD operations
* GUI-based management panels
* MVC-inspired architecture
* Modular manager classes

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* MVC Architecture
* Java Swing GUI
* Git & GitHub
* IntelliJ IDEA

## Project Structure

```text
src/
 ├── controller/
 ├── model/
 ├── view/
 └── Main.java
```

## Architecture

The project follows an MVC-inspired architecture:

* **Model** → Handles business entities and data management
* **View** → Contains GUI screens and user interaction panels
* **Controller** → Manages communication between views and models

This structure improves maintainability and code organization.
